servethis

A simple file sharing utility

Download Download View servethis on GitHub

servethis is a simple cross-platform file serving utility written in go. It provides a quick and easy way to share any files on your filesystem via http.

Installation

Assuming you have go installed you can simply run:
go get github.com/tehbilly/servethis.

If you do not have go installed you can download one of the pre-built releases and place the executable somewhere on your path.

Usage

Run servethis -h to see the plethora of options available to you!.

Without arguments servethis will serve the contents of the current working directory on port 8000. To change either the path that's served or the port that data is served on use the flags with the same names. For example: servethis -path c:/share/stuff -port 9191.
